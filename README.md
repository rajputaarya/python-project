# 💰 Smart Expense Splitter

A Python-based console application that helps users manage and split group expenses fairly among friends. This project allows users to create a group, add expenses, view all recorded expenses, calculate each member's balance, and generate settlements so everyone pays their fair share.

---

## 📌 Features

- Create a new expense group
- Add multiple group members
- Record expenses with:
  - Expense category (Food, Travel, Shopping, etc.)
  - Amount
  - Person who paid
- View all recorded expenses
- Calculate each member's share
- Display balances (who owes and who should receive money)
- Generate settlement suggestions

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Make sure Python 3 is installed.
4. Run the program:

```bash
python expense_splitter.py
```
